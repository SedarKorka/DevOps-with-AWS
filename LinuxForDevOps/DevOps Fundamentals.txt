## üêß Essential Linux Commands for DevOps

### Navigation
```bash
pwd                    # Print working directory
ls                     # List files
ls -l                  # Detailed list
cd path                # Change directory
cd ..                  # Parent directory
```

### File Management
```bash
cat file.txt           # Show file content
vim file.txt           # Edit file
rm file.txt            # Delete file
rm -r folder/          # Delete folder
```

### Package Management (Ubuntu/Debian)
```bash
sudo apt update                    # Update package list
sudo apt install service_name      # Install service
sudo apt remove service_name       # Remove service
```

### Package Management (CentOS/Amazon Linux)
```bash
sudo yum install service_name      # Install service
sudo yum remove service_name       # Remove service
```

### Network & Firewall
```bash
sudo ufw status                    # Firewall status
sudo netstat -tuln                 # Open ports
sudo netstat -tuln | grep 80       # Check port 80
```

### System Services (systemctl)
```bash
sudo systemctl status service      # Service status
sudo systemctl start service       # Start service
sudo systemctl stop service        # Stop service
sudo systemctl restart service     # Restart service
sudo systemctl reload service      # Reload configuration
```

### Nginx
```bash
# Installation
sudo apt install nginx             # Ubuntu/Debian
sudo yum install nginx             # CentOS/Amazon Linux

# Commands
nginx -v                           # Check version
sudo systemctl status nginx        # Nginx status
```
